# ud25_Almacenes
## API Almacenes

### CAJAS

#### GET
```
/api/cajas get All

/api/cajas/{num_ref}  get por numRef

/api/cajas/almacenes/{almacen}  get por Almacen
```

#### DELETE
```
/api/cajas/{num_ref} 
```

#### POST
```
/api/cajas/   RequestBody
```

#### PUT
```
/api/cajas/{num_ref}   RequestBody
```
